# Web UI (shadcn-style + HeroUI-inspired tokens)

## Getting Started

Run the dev server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

The backend (`server/`) can also serve the static export output (`./out`) at `http://localhost:8787/`.

### Static export

```bash
npm run build
```

This writes `./out` which is served by the backend when present.
